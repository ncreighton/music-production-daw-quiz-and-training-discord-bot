# Music Production & DAW Quiz and Training Discord Bot

Interactive quizzes, skill assessments, leaderboards, and certification tracking. Gamified learning for Music Production & DAW teams.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
